Security Auditor — Карманный сканер Linux-систем
Security Auditor — это легковесная утилита для быстрого поиска и автоматического исправления уязвимостей в ОС семейства Linux. Проект объединяет в себе функции аудита прав доступа, сетевой безопасности и анализа установленного ПО в едином веб-интерфейсе.  
📋 Основные возможности
Анализ прав доступа: Поиск файлов с правами 777 в критических директориях и проверка защищенности файла /etc/shadow.  
Сетевой аудит: Мониторинг открытых портов (TCP/UDP) и выявление потенциально опасных сервисов (FTP, Telnet, MySQL), доступных извне.  
Проверка пакетов: Сверка версий установленного ПО (через dpkg или rpm) с локальной базой известных уязвимостей CVE.  
Поиск утечек: Глубокое сканирование файловой системы на наличие забытых паролей, ключей и токенов по ключевым словам.  
Исправление в один клик: Генерация готовых команд для закрытия уязвимостей (остановка сервисов, блокировка портов через iptables).  
⚙️ Системные требования
Для корректной работы утилита использует нативные инструменты Linux, поэтому в системе должны быть установлены:  
Python: Версия 3.6 или выше.  
Сетевые утилиты: ss, ip, iptables.  
Менеджеры сервисов: systemctl.  
Пакетные менеджеры: dpkg или rpm.

1. Установка зависимостей Python:
Приложению требуется только микрофреймворк Flask для работы веб-интерфейса.  
pip3 install flask
2. Запуск сканера:
Для доступа к системным файлам и управления сетевыми правилами запуск должен производиться от имени суперпользователя.

 Архитектура
Single-File App: Весь бэкенд и фронтенд упакованы в один файл (~450 строк), что обеспечивает максимальную переносимость.  
Flask Backend: Обработка логики сканирования и выполнение команд через AJAX POST-запросы.  
Security First: Приложение по умолчанию привязывается к адресу 127.0.0.1, а команды с sudo исключены из кнопок авто-применения для защиты от случайного выполнения опасного кода через браузер.
